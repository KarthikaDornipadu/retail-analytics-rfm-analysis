# Retail Analytics using RFM Analysis

This project focuses on analyzing retail sales data to gain insights into customer purchasing behavior. The primary objective is to perform customer segmentation using the RFM model (Recency, Frequency, Monetary).

By analyzing transaction data, the project aims to identify different customer groups and understand purchasing patterns that can help businesses make better marketing and customer engagement decisions.

The project will utilize SQL for data querying, Python for data cleaning and analysis, and Power BI for visualizing insights through dashboards.

This repository has been created to organize the project files and manage team contributions. The project is currently in the initial stage and will be developed progressively as the analysis work begins.
